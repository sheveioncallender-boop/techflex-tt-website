# Installation / Upgrade

1. Commit the complete `techflex_tt_website` directory to the custom-addons repository used by CloudPepper.
2. Deploy/restart the Odoo service.
3. Open Apps and update the Apps list.
4. If the module is already installed, choose **Upgrade** for `TechFlex TT Complete Website`.
5. Hard-refresh the website or clear the browser cache so the v1.1.1 direct CSS/JS URLs load.

The module depends on `website_sale` and `mail` and supports Odoo 19 Community and Enterprise.
