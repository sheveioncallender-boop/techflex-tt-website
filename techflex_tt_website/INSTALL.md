# Installation and Configuration

## Compatibility

- Odoo 19 Community
- Odoo 19 Enterprise
- Module technical name: `techflex_tt_website`

## Installation

1. Copy `techflex_tt_website` into an Odoo custom addons path.
2. Restart Odoo.
3. Update the Apps list.
4. Install **TechFlex TT Complete Website**.

## Initial setup

### Products

1. Open a product template.
2. Publish the product on the website.
3. Open the **TechFlex Website** tab.
4. Set condition, warranty, highlights and specifications as needed.
5. Enable **Monthly Special** for products that should appear on the homepage.

### Product variants

Use standard Odoo attributes and variants for any customer-selectable option that changes price, SKU or stock, including RAM, storage, colour, size or warranty packages.

### Team and locations

Open **TechFlex Website → Website Content** to update the nine team cards and location information. Team images, names, roles, biographies and display order are editable.

### Website forms

Rental and contact submissions are saved under **TechFlex Website → Website Requests**. The records support status tracking and chatter/activity features.

## Deployment note

The package has been statically validated for Python syntax, XML structure, manifest paths and security CSV structure. A final installation and browser review should be performed on the target Odoo 19 staging database because this build environment does not include a running Odoo server.
