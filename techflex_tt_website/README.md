# TechFlex TT Complete Website for Odoo 19

Installable Odoo 19 Community and Enterprise website module for TechFlex TT Limited.

## Public pages
- Homepage
- Rentals request page
- About Us
- Contact Us
- Odoo native Shop and category routes with TechFlex visual styling
- Odoo native single-product page with TechFlex visual styling and additional product data

## Odoo functionality preserved
- Published products
- Website categories and routes
- Search, filters, sorting and pagination
- Product variants and attributes
- Pricelists, stock and sale pricing
- Cart, checkout, payment providers and portal

## Dynamic/custom data
- Homepage Monthly Specials use published Odoo products
- Product warranty, condition, highlights and technical specification lines
- Rental request records
- Contact request records
- Editable team members and locations

## Version 19.0.1.1.0
This release rebuilds the visual layer to use a wider 1600-1680px desktop presentation and more closely match the approved TechFlex page references.
