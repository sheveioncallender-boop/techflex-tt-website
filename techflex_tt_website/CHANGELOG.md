# Changelog

## 19.0.1.3.0

- Rebuilt the frontend as a single CSS source to remove conflicting legacy styles.
- Switched the approved heading typography to Barlow Condensed with Poppins body text and strong narrow-font fallbacks.
- Replaced skewed/low-resolution hero sources with clean wide hero assets for Home, Rentals, About, Contact, and Shop.
- Kept desktop two-column hero compositions down to tablet widths; mobile stacking starts below 760px.
- Expanded all page containers to the approved near-full-width layout.
- Restored category labels, service-panel copy, and denser homepage spacing.
- Enabled Odoo's native eCommerce category sidebar and set the website shop to four products per row.
- Strengthened Odoo 19 shop grid selectors while preserving native search, filters, sorting, categories, variants, cart, and checkout.
- Kept the product page inherited from Odoo 19 and removed the invalid legacy product-detail XPath.
